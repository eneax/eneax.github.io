CheckList

This project represents a basic checklist app built with CSS and Javascript. Type anything you want, then hit the submit button. A list will automatically be created. You can edit and delete the text fields previously added. Plus, you can also hide the elements that are not checked. Have a look!

Pig (dice game)

Pig is a simple game you can play with a single six-sided die. The numbers on the die, except for "1", represent points that the player can accumulate. The first player to score 100 or more points wins. Give it a try!
